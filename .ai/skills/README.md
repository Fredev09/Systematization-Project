# Skills Disponibles

Índice de todas las Skills de IA disponibles para el proyecto.

---

## Skills de la Comunidad (`skills/community/`)

### webapp-testing
- **Ruta**: `.ai/skills/community/webapp-testing/SKILL.md`
- **Origen**: [anthropics/skills](https://github.com/anthropics/skills)
- **Descripción**: Toolkit para testear aplicaciones web locales con
  Playwright. Incluye manejo del ciclo de vida del servidor, captura de
  screenshots y patrón de reconocimiento.

### frontend-design
- **Ruta**: `.ai/skills/community/frontend-design/SKILL.md`
- **Origen**: [anthropics/skills](https://github.com/anthropics/skills)
- **Descripción**: Guía de diseño visual distintivo para UI nueva o
  existente. Tipografía, paletas, layout, micro-copia y crítica de diseño.

### skill-creator
- **Ruta**: `.ai/skills/community/skill-creator/SKILL.md`
- **Origen**: [anthropics/skills](https://github.com/anthropics/skills)
- **Descripción**: Meta-skill para crear, modificar y optimizar Skills.
  Cubre el ciclo completo: captura de intención, escritura, evaluación
  y mejora iterativa.

## Skills del Proyecto (`skills/project/`)

### dynamic-forms
- **Ruta**: `.ai/skills/project/dynamic-forms/SKILL.md`
- **Descripción**: Especialista en la arquitectura EAV. DynamicService,
  Registro, ValorCampo, wrappers, hooks, fórmulas y validadores.

### migration
- **Ruta**: `.ai/skills/project/migration/SKILL.md`
- **Descripción**: Especialista en migraciones Legacy → Dynamic Forms.
  Sigue el patrón usado para migrar Ventas y Dashboard.

### django-review
- **Ruta**: `.ai/skills/project/django-review/SKILL.md`
- **Descripción**: Especialista en revisión de código Django. N+1,
  permisos, imports, URLs, vistas, templates, formularios, consistencia.

### ui-polish
- **Ruta**: `.ai/skills/project/ui-polish/SKILL.md`
- **Descripción**: Especialista en revisión visual. Espaciados,
  Bootstrap, responsive, accesibilidad, tablas, formularios, botones.

### documentation-sync
- **Ruta**: `.ai/skills/project/documentation-sync/SKILL.md`
- **Descripción**: Especialista en mantener documentación sincronizada.
  Actualiza SESSION_LOG.md, TODO.md, MIGRATION_STATUS.md, DECISIONS.md.

---

## Formato de registro

Cuando se agregue una Skill, documentarla así:

```markdown
### [Nombre de la Skill]
- **Ruta**: `.ai/skills/{tipo}/nombre-skill.md`
- **Descripción**: Breve descripción de qué hace.
- **Tarea asociada**: Enlace a la tarea en `docs/TODO.md` si aplica.
```
